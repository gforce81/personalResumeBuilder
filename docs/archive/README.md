## Files in Archive

Migration notes from v1.0 (email/password) to v1.1 (Google Sign-in).

**Why archived:** Google Sign-in is now the default authentication method. New users don't need migration instructions.

**Relevant for:** Understanding the evolution of the authentication system.

### ADMIN_SECURITY_IMPLEMENTATION.md
Implementation details of the admin email restriction feature from v1.2.

**Why archived:** The information is now consolidated in ADMIN_EMAIL_RESTRICTION.md which is more user-friendly and up-to-date.

**Relevant for:** Understanding the technical implementation decisions.
